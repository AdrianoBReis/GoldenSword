# Godot 3D Arena Showcase

Protótipo 3D feito em Godot para demonstrar recursos de:
- movimentação de personagem em `CharacterBody3D`;
- iluminação + céu procedural;
- física e colisão em 3D;
- interação com raycast (disparo em alvos);
- HUD com pontuação e cronômetro.

## Como executar
1. Abra o projeto no Godot 4.2+.
2. Rode a cena principal `scenes/Main.tscn`.

## Controles
- **WASD**: mover
- **Mouse**: mirar
- **Botão esquerdo**: atirar
- **Espaço**: pular
- **Esc**: liberar mouse

## Objetivo
Colete os discos azuis e acerte alvos vermelhos antes do tempo acabar.
