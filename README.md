# Godot 2D Neon Arena Showcase

Protótipo 2D feito em Godot para demonstrar recursos de alto nível:
- movimentação com aceleração, atrito e dash em `CharacterBody2D`;
- raycast 2D para tiro com feedback visual imediato;
- iluminação dinâmica com `PointLight2D` e atmosfera neon;
- coleta, alvos móveis, combo e pontuação reativa;
- HUD com cronômetro, multiplicador e reinício rápido.

## Como executar
1. Abra o projeto no Godot 4.2+.
2. Rode a cena principal `scenes/Main.tscn`.

## Controles
- **WASD**: mover
- **Mouse**: mirar
- **Botão esquerdo**: atirar
- **Espaço**: dash
- **Esc**: mostrar cursor

## Objetivo
Colete núcleos de energia e destrua drones neon antes do tempo acabar.
